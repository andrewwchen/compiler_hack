# Part 3: Optimization

## Usage
first write a mini-c program to optimize into opt_tests/test.ll
```
make
make llvm_file
./llvm_parser opt_tests/test.ll
```
results in optimized test file test_new.ll
```
make clean
```

## Files
```
/opt_tests - provided tests
Core.h - provided
llvm_parser.c - main c file for optimizer
```
